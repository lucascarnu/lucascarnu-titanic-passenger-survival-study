# Titanic Passenger Survival Analysis

## Project Overview
This project explores the Titanic passenger dataset using Python and data analysis libraries. The goal is to examine patterns that may help explain why some passengers survived while others did not.

## Dataset Description
The dataset contains information about passengers aboard the Titanic, including demographic data, ticket class, fare paid, and survival outcome.

## Exploratory Analysis
Initial exploration was performed to understand the dataset structure, including:

- inspecting rows and columns
- reviewing data types and statistics
- identifying missing values
- checking duplicated records

These steps helped provide a clearer understanding of the dataset before performing deeper analysis.

## Observations

Some patterns observed during the analysis include:

- Female passengers had a significantly higher survival probability compared to males.
- Passengers traveling in first class tended to survive more frequently than those in lower classes.
- Younger passengers, particularly children, showed higher survival rates.
- Most passengers were between roughly 20 and 40 years old based on the age distribution.

## Tools and Libraries
The analysis was conducted using:

- Python
- Pandas
- Matplotlib

## Final Notes
The results illustrate how social factors such as class, gender, and age influenced survival during the Titanic disaster. This project demonstrates basic exploratory data analysis and visualization techniques using Python.
